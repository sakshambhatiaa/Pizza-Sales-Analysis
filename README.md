# 🍕 Pizza Sales Analysis using SQL

## 📌 Overview

This project analyzes a pizza restaurant's sales data using SQL to extract valuable business insights. The analysis focuses on sales performance, customer ordering patterns, pizza popularity, revenue distribution, and product performance.

The project demonstrates practical SQL skills by solving real-world business problems using joins, aggregations, subqueries, and window functions.

---

**Dataset Source:**  
https://www.kaggle.com/datasets/shilongzhuang/pizza-sales

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing pizzas.
- Understand customer ordering behaviour.
- Evaluate category-wise sales contribution.
- Generate business insights through SQL queries.

# 📊 Key Performance Indicators (KPIs)

### Sales Performance

- Total Number of Orders
- Total Revenue
- Average Pizzas Ordered per Day

---

### Product Analysis

- Highest Priced Pizza
- Lowest Priced Pizza
- Top 5 Most Ordered Pizzas
- Top 3 Revenue-Generating Pizzas
- Most Common Pizza Size Ordered

---

### Category Analysis

- Total Quantity Sold by Category
- Category-wise Pizza Distribution
- Revenue Contribution by Category
- Top 3 Revenue-Generating Pizzas in Each Category

---

### Customer Ordering Behaviour

- Order Distribution by Hour
- Daily Ordering Trends
- Average Daily Quantity Ordered
